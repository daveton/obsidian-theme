# Bear Theme for Obsidian

A beautiful Obsidian theme inspired by the elegant design of [Bear App](https://bear.app). This theme faithfully recreates the visual experience of Bear's iconic **Red Graphite** (light) and **Charcoal** (dark) themes.

## Features

- **Two Beautiful Themes**: 
  - **Red Graphite** - Clean light theme with crisp white backgrounds and distinctive red accents
  - **Charcoal** - Sophisticated dark theme with deep gray backgrounds and vibrant red highlights

- **Bear-Style Typography**: 
  - System font stack optimized for macOS, iOS, and cross-platform readability
  - Carefully tuned font sizes and line heights matching Bear's elegant spacing
  - SF Pro-inspired heading hierarchy

- **Thoughtful UI Elements**:
  - Custom checkbox styling inspired by Bear's checkbox SVG icons
  - Beautifully styled tags with rounded borders
  - Clean blockquotes with signature red left border
  - Polished code blocks with syntax highlighting colors

- **Perfect Spacing**: Generous margins and padding for comfortable reading

## Installation

### Method 1: Community Themes (Recommended)
1. Open Obsidian Settings
2. Go to **Appearance** → **Themes**
3. Click **Manage** and search for "Bear"
4. Click **Install** and then **Use**

### Method 2: Manual Installation
1. Download the `theme.css` and `manifest.json` files from the `obsidian-ui/` folder
2. Create a folder named `bear` in your Obsidian vault's `.obsidian/themes/` directory
3. Copy `theme.css` and `manifest.json` into the `bear` folder
4. In Obsidian, go to **Settings** → **Appearance** → **Themes**
5. Select "Bear" from the dropdown menu

## SVG Resources

This theme incorporates SVG icons extracted from Bear App resources:

- **External Link Arrow** (`right arrow attachment.svg`) - Indicates external links
- **Map Pin** (`map attachment.svg`) - Indicates location/map links
- **Checkbox Style** - Based on `checkbox attachment.svg`

All icons are embedded as data URIs in the CSS for zero external dependencies.

- **Interface**: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto`
